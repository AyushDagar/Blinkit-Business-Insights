## üìå Project Overview
The goal of this project was to transform raw Blinkit sales data into a clean, interactive, and visually intuitive dashboard using Microsoft Excel. The dashboard allows users to:
- Track **Total Sales**, **Average Sales**, **Number of Items**, and **Average Rating**
- Compare outlet performance by **Size**, **Location**, and **Establishment Year**
- Analyze product behavior based on **Item Type**, **Fat Content**, and **Category**
- Explore trends and insights using slicers and dynamic filters

Everything from data cleaning to visualization was done inside Excel using pivot tables, charts, lookup sheets, and custom formatting.

## üìä Key Insights from the Data
A few interesting insights the dashboard reveals:
- **Regular-fat products generate more revenue** than low-fat items  
- **Tier 2 outlets dominate sales**, outperforming Tier 1 and Tier 3 combined  
- **High outlet size locations show the strongest revenue figures**  
- Product categories like *Snack Foods* and *Fruits & Vegetables* significantly outperform others  

Each chart and pivot table contributes to building a clear narrative around Blinkit's performance.
